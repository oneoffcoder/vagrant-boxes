# Purpose

A vagrant box with

* Docker (latest)
* Miniconda v3 with Python 3.7
* OpenJDK v1.8
* Maven v3.6.1
* Ant v1.9.14
* SBT v1.2.8
* Node v12.9.1

# Use

To bring up.

```bash
vagrant up
```

To destroy.

```bash
vagrant destroy -f
```

To SSH into the VM. The login/password is `vagrant/vagrant`.

```bash
vagrant ssh
```
